路由分发

【一】什么是路由分发

  • 路由分发（Route Redistribution）是指在一个网络环境中，当存在多种路由协议或同种协议的不同进程时，为了实现这些路由协议的协同工作，通过自治系统边界路由器（ASBR）同时工作在两种协议或两个进程中，学习到两端所有的路由条目后，将这些路由信息从一个路由协议或进程共享到另一个路由协议或进程的过程

【二】路由分发系统解析

  • 就是在每个APP下都有自己的路由映射规则文件

  • 将所有的给规则文件进行统合到一起

虚拟环境

【一】创建虚拟环境

【1】三种方法

  • python自带你的venv工具
  • python第三方模块virtualenv

  • 使用pycharm创建虚拟环境

【二】使用虚拟环境的目的

  • 目的就是：为了项目与项目之间模块的隔离
  • 不通过版本写的Django项目，项目之间会发生冲突，不同版本的Django，也会对应MySQL的版本，有的Django版本不兼容MySQL
    • 为了解决这个问题，给各自的Django项目创建虚拟环境
    • 但是有个弊端，你自己的Django项目本地随便用
    • 你压缩给别人用，别人解压之后虚拟环境就会失效

【1】解决不兼容虚拟环境的通用方法

    • 导出成功

（2）批量安装

  • 在终端输入

  • 路径转换器（Path Converter）在Django框架中是为了解决URL路径中的参数问题

【三】Django的五种类型转换器

【1】str

  • 匹配除了/之外的非空字符串
  • 如果表达式内不包含转换器，则会默认匹配字符串

【2】int

  • 匹配 0 或任何正整数。返回一个 int

【3】slug

  • 匹配任意由 ASCII 字母或数字以及连字符和下划线组成的短标签
  • 比如，building-your-1st-django-site

【4】path

  • 匹配非空字段，包括路径分隔符 ‘/’
  • 它允许你匹配完整的 URL 路径而不是像 str 那样匹配 URL 的一部分

【5】uuid

  • 匹配一个格式化的 UUID 。为了防止多个 URL 映射到同一个页面，必须包含破折号并且字符都为小写
  • 比如，075194d3-6885-417e-a8a8-6c931e272f00。返回一个 UUID 实例
