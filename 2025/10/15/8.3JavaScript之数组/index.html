JavaScript之数组

【一】什么是数组

  • 数组是一组有序的数据集合，但它不是python中的集合，但它类似于python中的列表
  • 数组内部可以存放多个数据，可以存放任意类型，数据的长度可以动态调整

【二】创建数组

存放多个数据时用，隔开

  • 通过字面量的方式创建数组
```javascript var arr = [];

【三】数组获取元素

  • 索引获取元素，索引从0开始
```javascript var numList=[1,2,3,4,5]; numList[0]; 1 numList[4]; 5

【五】数组的常用方法

方法名称 功能描述 返回值类型 是否改变原数组
forEach/for…of 遍历数组（forEach处理每个元素，for…of可读取元素并自动解包） 无 不会
concat 合并数组 新数组 不会
join 将数组转化为字符串 字符串 不会
pop 删除数组最后一个元素 删除的数据（即数组最后一个元素） 会
shift 删除数组第一个元素 删除的数据（即数组第一个元素） 会
unshift 在数组开头插入元素 新数组长度 会
push 在数组末尾添加元素 新数组长度 会
reverse 翻转数组顺序 翻转后的数组 会
sort 对数组元素排序 排序后的数组 会
slice 截取数组部分元素 新数组 不会
splice 插入、删除或替换数组元素 被删除或替换的元素 会
toString 数组转为字符串 字符串 不会
valueOf 获取数组原始值 原始值 不会
indexOf 查找指定元素在数组中的位置 元素索引（若不存在则返回 -1） 不会
lastIndexOf 查找指定元素在数组中最后一次出现的位置 元素索引 不会
map 应用函数映射数组元素 新数组 不会
filter 根据条件筛选数组元素 新数组 不会
every 判断数组所有元素是否都满足条件 布尔值 不会
some 判断数组是否有元素满足条件 布尔值 不会
reduce 按规则累积数组元素生成单个返回值 单一返回值 可以，但通常不会直接改变原数组
rightReduce 类似于reduce，但从数组末尾开始累积 单一返回值 可以，但通常不会直接改变原数组

【1】方法使用

（1）forEach方法

遍历数组，遍历每个元素

  • value：数组里的每个元素
  • index：数组中每个元素的索引
  • 两种方式
    • =>：叫箭头函数
    • function(){}，普通函数
```javascript // 方式一：=> var numList=[1,2,3,4,5]; numList.forEach((value,index)=>{ console.log(`value:${value}`) console.log(`index:${index}`) }); value:1 index:0 value:2 index:1 value:3 index:2 value:4 index:3 value:5 index:4 // 方法二：function var numList=[1,2,3,4,5]; numList.forEach(function(value,index){ console.log(`value:${value}`) console.log(`index:${index}`) }); value:1 index:0 value:2 index:1 value:3 index:2 value:4 index:3 value:5 index:4

（2）for…of方法

  • 读取元素并自动解包
```javascript var numList=[1,2,3,4,5]; for(var item of numList){ console.log(item) }; 1 2 3 4 5

(4)join方法

  • 将数组转为字符串,按照指定的字符拼接
```javascript var numOne=[1,2,3,4,5]; // 接数组，返回字符串 var str=numOne.join("￥"); str '1￥2￥3￥4￥5'

（6）unshift方法

  • 插入数据，数据插入到数组的开头
    • 只能添加到数组
```javascript var arr=[2,3,4]; arr.unshift("3"); 4 // 返回的是数组的长度 arr ['3', 2, 3, 4]

（8）reverse方法

  • 反转数组顺序
```javascript var arr=[2,3,4]; arr.reverse(); [4, 3, 2]

(18)every方法

  • 判断数组里的所有元素是否满足条件
```javascript var arr=[0,2,true,false]; arr.every(val=>val); false var arr=[4,2,true,3]; arr.every(val=>val); true

(20)reduce方法

  • 累积数组元素生成单个返回值
    • total：累加器的当前值
    • num：正在处理的数组元素
```javascript const arr = [1, 2, 3, 4, 5]; const sum = arr.reduce((total, num) => total + num, 0); // 这个0代表初始值是0，要不设置这个的话初始值就会是1 console.log(sum); // 输出：15(等于sum)

（21）reduceRight方法

  • 和reduce方法很像
  • 但是rightReduce方法是从后向前累加
    • total：累加器的当前值
    • num：正在处理的数组元素
```javascript let arr = [1, 2, 3, 4, 5]; let reversedSum = arr.reduceRight((total, num) => total + num, 0); console.log(reversedSum);// 15 (等于sum，因为是从后向前累加)

（22）统计数组的长度

  • 用方法length
```javascript var arr=[1,2,3,4,5,6,7]; arr.length; 7

【2】两个参数的时候

  • value：代表数组中的每个值
  • index：代表数组中每个值的索引值
```javascript var arr=[1,2,3,4,5,6]; arr.forEach(function(value,index){ console.log(`这是value---${value}`) console.log(`这是index---${index}`) }); // 这是value---1 // 这是index---0 // 这是value---2 // 这是index---1 // 这是value---3 // 这是index---2 // 这是value---4 // 这是index---3 // 这是value---5 // 这是index---4 // 这是value---6 // 这是index---5

【七】splice

【1】两个参数的时候

  • splice：第一个是参数是，索引起始位置，第二个参数是删除的个数
```javascript var arr=[1,2,3,4,5,6]; console.log(arr.splice(1,3)); [ 2, 3, 4 ] console.log(arr); [ 1, 5, 6 ]

【八】map

  • 类似于forEach方法
```javascript var arr=[1,2,3,4,5,6]; var arrOne= arr.map(function(value,index){ console.log(value) console.log(index) return value // 可以返回一些基于value的新值 }) console.log(arrOne) // 1 // 0 // 2 // 1 // 3 // 2 // 4 // 3 // 5 // 4 // 6 // 5 // [ 1, 2, 3, 4, 5, 6 ]
