JavaScript之对象

【一】对象

  • javascript中对象是无序的属性集合
  • 我们可以把JavaScript中的对象看成键值对，其中之可以是数据或者函数
  • 在对象中属性表示：特征
  • 在对象中方法表示：行为

【二】对象的创建

【1】定义对象

  • 方式一
```javascript var obj={};

【2】构造函数

  • 自定义一个创建具体对象的构造函数，函数内部不需要new构造的过程，直接使用this代替对象进行属性和方法的书写，也不需要return一个返回值
  • 使用时利用new关键字调用自定义的构造函数就可以了
  • 一般构造函数首字母需要大写
```javascript function Obj(name, age) { this.name = name; this.age = age; } // 使用 new 关键字创建一个新的 Obj 对象 var person = new Obj("Alice", 30); // 现在 person 是一个对象，它有两个属性：name 和 age console.log(person.name); // 输出 "Alice" console.log(person.age); // 输出 30

【2】方法调用

  • 函数名.方法名===>objectName.methodName()
```javascript let Person={ name:"tom", age:18, getAge:function(){return this.age} } console.log(Person.getAge()); // 18

【4】for循环遍历

```javascript const person = { name: "John Doe", occupation: "Software Engineer", getOccupation: function() { return this.occupation; } }; for (let key in person) { if (person.hasOwnProperty(key)) { console.log(`${key}: ${person[key]}`); } } // name: John Doe // occupation: Software Engineer // getOccupation: function() { return this.occupation; }

（2）创建指定的日期和时间的对象实例

  • specific：具体的
```javascript var specficDate=new Date(2024,6,12,19,53); console.log(specficDate); Fri Jul 12 2024 19:53:00 GMT+0800 (中国标准时间)

（4）转换时间格式–>toLocaleString

```javascript var currentDate=new Date(); console.log(currentDate.toLocaleString()); 2024/6/12 19:58:53

【2】Date的内置方法

（1）获取日期和时间信息

  • getFullYear()–>年份

  • getMonth()–>月份，0表示1月，11表示12月

  • getDate()–>日，月中的某一天

  • getDay()–>获取星期

  • getHoure()–>获得小时

  • getMinutes()–>获得分钟

  • getSeconds()–>获得秒数

  • getMillseconds()–>获得毫秒数

  • getTime()–>获得时间戳

（2）案例

```javascript var time=new Date(); console.log(time.getFullYear()) console.log(time.getMonth()) console.log(time.getDate()) console.log(time.getDay()) console.log(time.getHours()) console.log(time.getMinutes()) console.log(time.getSeconds()) console.log(time.getMilliseconds()) console.log(time.getTime()) // 2024 // 5 // 12 // 3 // 20 // 14 // 34 // 541 // 1718194474541

【五】JS中的序列化

【1】python中

  • dumps：序列化
  • loads：反序列化

（1）例子

```python import json # 示例：创建一个Python字典 data = { "name": "John Doe", "age": 30, "is_student": False, "hobbies": ["reading", "coding"], } # 序列化为JSON字符串 json_string = json.dumps(data, indent=4) # 添加缩进以提高可读性 print(json_string,type(json_string)) # { # "name": "John Doe", # "age": 30, # "is_student": false, # "hobbies": [ # "reading", # "coding" # ] # } parsed_data = json.loads(json_string) print(parsed_data,type(parsed_data)) # {'name': 'John Doe', 'age': 30, 'is_student': False, 'hobbies': ['reading', 'coding']}

（2）反序列化（parse）

  • JSON.parse() 函数可以将JSON格式的字符串解析回JavaScript对象或值。
```javascript var data={ "name":"tom", "age":18 } var dataStr=JSON.stringify(data); console.log(dataStr,typeof dataStr); // 输出：{"name":"tom","age":18} string var dataObj=JSON.parse(dataStr); console.log(dataObj); // 输出：[name: 'tom', age: 18} 'object'
  • 方式二
```javascript let reg=/正则表达式/

（2）exec()

  • 字符串中搜索匹配正则表达式的内容，并返回结果数组。
    • 如果没有则返回null
    • pattern：模式
```javascript var str = "Hello, world!"; var pattern = /l+/g; var result = pattern.exec(str); console.log(result); // 输出：['ll', index: 2, input: 'Hello,World!', groups: undefined]

（4）search()

  • 在字符串中搜索匹配正则表达式的内容，并返回第一个匹配的索引位置。
    • 如果没有匹配的内容，则返回-1
```javascript var str = "Hello, world!"; var pattern = /lo/; var result = str.search(pattern); console.log(result); // 3

（6）flag()

  • 返回正则表达式的修饰符标志字符串
```javascript var patternFive = /Hello/gi; var patternSix = /Hello/g; console.log(patternFive.flags); // gi console.log(patternSix.flags); // g

【4】正则的BUG

（1）全局模式的指针移动

```javascript let reg = /^[a-zA-Z][A-Za-z0-9]/g // 第一次匹配成功 -- 有数据-- 指针移动到尾端 reg.test("tom"); // true // 第二次匹配失败 -- 指针在尾端向后匹配 --无数据 // 匹配失败会回到开头保证下次，匹配成功 reg.test("tom"); // false // 第三次匹配成功 -- 有数据-- 指针回到头部 reg.test("tom"); // true reg.test("tom"); // false // 第二次匹配失败 -- 指针在尾端向后匹配 --无数据 reg.lastIndex // 0 reg.test("tom"); // true // 第三次匹配成功 -- 有数据-- 指针回到头部 reg.lastIndex // 2

【七】math对象

  • 数学对象

【1】方法

```javascript abs(x) 返回数的绝对值。 exp(x) 返回 e 的指数。 floor(x) 对数进行下舍入。 log(x) 返回数的自然对数（底为e）。 max(x,y) 返回 x 和 y 中的最高值。 min(x,y) 返回 x 和 y 中的最低值。 pow(x,y) 返回 x 的 y 次幂。 random() 返回 0 ~ 1 之间的随机数。 round(x) 把数四舍五入为最接近的整数。 sin(x) 返回数的正弦。 sqrt(x) 返回数的平方根。 tan(x) 返回角的正切。

（2）exp(x)

  • 返回 e (自然对数的底数，约为 2.71828) 的 x 次方。x 必须是一个数字类型
```javascript let exponent = 3; console.log(Math.exp(exponent)); // 输出: 20.085536923187668 (近似值)

（4）log(x)

  • 返回传入参数 x 的自然对数（底为 e）。x 必须大于 0，并且是一个数字类型。
```javascript let baseTenNum = 100; let a = Math.log(baseTenNum); console.log(a); // 输出: 4.605170185988092 (近似值，因为 ln(100) ≈ 4.605)
