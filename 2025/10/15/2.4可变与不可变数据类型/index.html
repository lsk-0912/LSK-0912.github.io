可变与不可变数据类型

#ID如果发生变化就是不可变数据类型

#ID如果不变就是可变数据类型

【一】可变数据类型

  • id 不变是可变数据类型

（1）列表

  • 增
    • append
    • extend
    • insert
  • 删
    • pop（）默认弹出最后一个
    • remove（）按照索引位置插入
  • 改
    • 索引替换值
  • 查
    • 直接索引查询

（2）字典

  • 增
    • [key] = value
    • setdefault(key,value)有返回值
    • .update(k=v)
  • 删
    • pop（k）
    • popitem()随即删除一对键值对
  • 改
    • 变量名[k]=要改的v
  • 查、
    • [k] 直接取键，取不到会报错
    • .get(k)取不到返回None,不会报错

（3）集合

  • 无需去重

【二】不可变数据类型

【1】字符串

  • 不能索引改值可以索引取值
  • 拼接用+
  • 切片（顾头不顾尾）
    • [1:4]从索引1到4不包括4
    • 步长[1:4:2]从索引1到索引4（不包括4） 步长为 2
    • 计算长度len
  • 去除空格
    • strip() 方法会去除字符串开头和结尾的所有空格。
    • lstrip() 方法用于去除字符串开头的空格。
    • rstrip() 方法用于去除字符串结尾的空格。
  • 切片
    • split（“|”）默认是从左向右切
    • rsplit（“|”）从右向左切
  • 遍历
    • for
    • whlie
  • 大小写字母转换
    • 小写转大写(upper())
    • 大写转小写(lower())
  • 首尾字符判断
    • 判断字符开头(startswith())
    • 判断字符结尾(endswith())
  • 格式化输出
    • %s%d
    • format（）+{}
    • f+{需要添加的变量}
  • 替换
    • 以旧换新replace（旧字符串，新字符串）
  • 判断类型isdigit
    • 判断字符串是否是纯数字组成，返回结果为True或False
  • 添加
    • " ".join( )
  • 填充
    • o填充 ljust()
    • 两端填充print(变量名.center(8,‘-’))
    • 左对齐填充左边print(name.rjust(7,‘*’))
    • 右对齐填充右边print(name.ljust(7,‘*’))
  • 查
    • index print(name.index(‘x’)) 查不到就会报错
    • find print(name.find(‘x’))查不到会返回-1

【2】数字类型

  • 判断数字类型(isdigit)
  • 十进制转换

【3】元组

  • print((1, 2, 3) * 3)元组重复

【4】布尔

  • 真和假（True，False）
