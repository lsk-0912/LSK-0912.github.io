Django安装

【一】建议

  • 首先要知道自己的环境变量
  • 区分我们现在的python版本
    • 根据你在环境变量里面python解释器的定义顺序决定的
    • 那个解释器在上默认就是那个解释器
  • 默认是那个解释器pip安装Django就安装在那个解释器上
  • pip install django

【二】安装Django注意事项

  • 安装Django的时候，计算机名称不建议设置成中文
  • python解释器不同可能会引起报错
  • Django项目名和路径名不能出现中文---->****
  • 一个Django项目是一个单独的项目---->****
    • 不要出现其他无关紧要的文件或文件夹，都不可以

【三】Django安装

【1】安装命令

  • 直接insatall回安装最新版本的Django
```shell pip install django

  • 安装指定的Django的版本
  • 比如你安装的有Django它会自动把你的删掉，然后把你指定的下载上
```shell pip install django==3.2.12

【四】如何验证

  • 下面这句话代表安装成功
```shell Successfully installed asgiref-3.8.1 django-3.2.12 pytz-2024.1 sqlparse-0.5.0 typing-extensions-4.12.2

【2】直接去指定的文件目录下找

  • 在python解释器下的Scripts文件夹下

（1）创建目录

（4）启动Django项目

  • 前提是你要确保你在创建好的django文件夹下

【3】注册APP使用APP

（1）注册APP

  • settings.py
    • 在pycharm里面创建的app自动注册app
```python INSTALLED_APPS = [ 'django.contrib.admin', 'django.contrib.auth', 'django.contrib.contenttypes', 'django.contrib.sessions', 'django.contrib.messages', 'django.contrib.staticfiles', "user1.apps.User1Config" # 必须注册app才能使用app 'user' ]
  • 第三步 要在路由映射关系 first_django/urls.py 注册映射关系
```python from django.contrib import admin from django.urls import path from user.views import login urlpatterns = [ path('admin/', admin.site.urls), # 默认访问根目录的时候 会执行 login 视图函数返回数据 path('', login), ] ```
