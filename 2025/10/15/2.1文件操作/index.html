文件操作

【一】打开文件的两种方式（open和with open）

【1】有三种模式(r,w,a)

（1）r : 只读不取（read）

  • /格式：
```python #通过with open ......语句打开文件，并将文件内容一次性读完 with open（'这个是你创建的文本地址'，**mode =** "r",**encoding =** "utf-8"） as l:#l这个值可以随便取值 #会将原本的内容由硬盘全部读入内存，赋值给res res = l.read() print（res）#你文本里面的信息就会被读取 #l.close()执行这个语句是将你创建的文本文件关闭 #它还可以进行切片 #文件0.1txt里面是tom|710 with open("0.1txt","r",encoding = "utf-8") as ly: num = ly.read() num_one = num.split("|") print(num_one)#['tom', '710']

（2）w : 只写（write）

  • 在文件不存在时会创建空文档，文件存在时会清空文件，文件指针跑到文件开头
```python #写入一次 data = "name|912" with open(file="0.1txt", mode="w", encoding="utf-8") as ly: ly.write(data)#文本里就是name|912 #连续写入 data = "name|912" age = "18" with open(file="0.1txt", mode="w", encoding="utf-8") as ly: ly.write(data ) ly.write(age)#文本里就是name|91218 #（）括号里面也可以直接输入文字但是要带引号
  • 文件存在则打开并继续写入（文件指针直接移动到末尾，不会清空原来的文本信息）
```python name=input('username>>>: ').strip()#输入名字 pwd=input('password>>>: ').strip()#输入密码 with open('0.1.txt',mode='a',encoding='utf-8') as f: info='%s:%s\n'%(name,pwd)#%s占位，你输入的名字会保存在0.1txt中 f.write(info)#保存

（4）a和w的区别

  • a文本信息添加的时候不会吧原有的清除
  • b文本就会把原有的文本信息清除
  • 相同点就是文本信息打开时，都可以连续写入

（5）+模式的应用

  • r+，w+，a+ : 可读可写
  • 用的最多的就是rwa。

【二】文件读写内容模式

  • 前提是tb不能单独使用，必须与r/w/a之一结合使用

【1】t : 文本模式

  • 读写的文件都是以字符串为单位的，只针对文本文件，必须指定encoding参数

（1）使用：

  • 如果指定的文件打开模式为r/w/a，其实默认的就是rt/wt/at
```python name_list = ['dream', 'hope'] with open('01.txt', 'w', encoding='utf8') as fp: # 无法直接写入 列表 / 字典等类型 # fp.write(name_list) # Expected type 'str' (matched generic type 'AnyStr'), got 'list[str]' instead # 只能写入字符串格式的内容，可以使用str强转 fp.write(str(name_list)) #文件夹里面有文本的话覆盖在输入 #想连续的话就在fp.write(str(name_list))在加一个 #例子 with open('0.1.txt', 'r', encoding='utf-8') as fp: data = fp.read()#如果文本0.1.txt里面有值的话 # 通过 r 模式读出来的数据都是字符串格式的 print(data, type(data)) #文本0.1.txt里面的内容就是 欧里给 # 及时读出来的数据符合列表格式，强转以后，仍然是字符串格式 data_list = list(data) print(data_list, type(data_list)) #['欧', '里', '给']
  • 针对非文本文件（图片、视频、音乐等）只能使用b模式。
  • rb（read bytes）
  • ab(append bytes)
  • wb(write bytes)

【三】文件的操作方法

读【1】读操作

  • f.read()
    • 读取所有内容，执行完该操作后，文件移动到文件末尾
  • f.readline()
    • 读取一行内容，光标移动到第二行首部
  • f.readlines()
    • 读取每一行内容，存放于列表中

（1）一次性读完方式一

```python #循环遍历 with open("0.2txt",mode="rt",encoding="utf-8") as ly: for name in ly: print(name)#一次只读取0.2txt文本的一行内容 # with open("0.2txt",mode="rt",encoding="utf-8") as ly: while True: data =ly.readline() if data: print( data) else: break

写【2】写操作

  • f.write()
    • 一次性全部写入
  • f.writelines()
    • 可以迭代写入可迭代对象的内容

(1)write方法

```python name = ["tom","710","/n","dream","123","/n"] with open("01.txt",mode="a",encoding="utf8") as lm: lm.write(str(name)) # 虽然01.txt文本文件夹里是["tom","710","/n","dream","123","/n"]这个，但是它是字符串

【3】判断文件是否可读/可写/可关闭、

  • f.readable()

    • 文件是否可读

  • f.writable()

    • 文件是否可写

  • f.close()

    • 文件是否可关闭

  • f.encoding()

    • 如果文件打开模式为b，则没有该属性

  • f.flush()

    • 立刻将文本内容从内存刷到硬盘

  • f.name()

    • 获取文件名字
```python with open('example.txt', mode='w', encoding='utf-8') as f: print(f"可读性: {f.readable()}") # 输出: 可读性: False print(f"可写性: {f.writable()}") # 输出: 可写性: True print(f"是否关闭: {f.closed}") # 输出: 是否关闭: False print(f"编码格式: {f.encoding}") # 输出: 编码格式: utf-8 f.flush() # 立即将内容刷新到硬盘 print(f"在 'with' 块之后是否关闭: {f.closed}") # 输出: 在 'with' 块之后是否关闭: True

【1】seek函数

  • f.seek(指针移动的字节数，模式控制)：
  • 0：默认的模式，该模式代表指针的字节数是以文件开头为参照物的
  • 1：指针的位置是以当前为参照物（py3.8以后好像就不允许操作文本了）
  • 2：指针的位置是以末尾为参照物的
  • 模式1：以当前位置为参照物向后移动
  • 模式2：以结尾位置为参照物移动
  • lm.seek(-2,2)
  • 字节数：字母占一个字节，汉字占三个字节
