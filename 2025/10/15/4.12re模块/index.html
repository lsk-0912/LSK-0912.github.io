re模块

【一】正则语法

  • 在线测试工具 http://tool.chinaz.com/regex/

  • 正则就只和字符串相关

  • 正则：按照指定的匹配规则从字符串中匹配或者截取相应的内容

【1】字符组

  • [字符组] 在同一个位置可能出现的各种字符组成了一个字符组
  • 字符组用[]
  • 字符组的[备选项]字母/数字/特殊标点…
  • 语法re.findall(正则模式，文本)，基于正则模式查找所有匹配的文本内容
正则 待匹配字符 匹配结果 说明
[0123456789] 8 True 在一个字符组里枚举合法的所有字符，字符组里的任意一个字符和"待匹配字符"相同都视为可以匹配
[0123456789] a False 由于字符组中没有"a"字符，所以不能匹配
[0-9] 7 True 也可以用-表示范围,[0-9]就和[0123456789]是一个意思
[a-z] s True 同样的如果要匹配所有的小写字母，直接用[a-z]就可以表示
[A-Z] B True [A-Z]就表示所有的大写字母
[0-9a-fA-F] e True 可以匹配数字，大小写形式的a～f，用来验证十六进制字符
  • 例题：re.findall(正则模式，文本)
```python import re ret2=re.findall("a","asdfgaa") print(ret2)#['a', 'a', 'a'] ret3=re.findall("[4,6]","ad,gh,54,14") print(ret3)#[',', ',', '4', ',', '4'] ret4=re.findall("[ace]", "a,b,c,d,e") print(ret4)#['a', 'c', 'e'] #匹配a~z小写字母 ret5 = re.findall("[a-z]", "a,b,c,d,e") print(ret5)#['a', 'b', 'c', 'd', 'e'] #匹配0~9数字 ret6 = re.findall("[0-9]", "1,2,3,4,5") print(ret6)#['1', '2', '3', '4', '5'] #除了字母以外 ret7 = re.findall("[^a-z]", "1,a,2,b,3") print(ret7)#['1', ',', ',', '2', ',', ',', '3'] ret8 = re.findall("\d", "1,2,3,4,5") print(ret8)#['1', '2', '3', '4', '5'] ret9 = re.findall("[0-9a-z]", "1,a,2,b,3") print(ret9)#['1', 'a', '2', 'b', '3']

【3】重复元组字符-----》 +* {} ?

```python ret2 = re.findall("[0-9a-zA-Z]", "apple,banana,orange,melon")#['a', 'p', 'p', 'l', 'e', 'b', 'a', 'n', 'a', 'n', 'a', 'o', 'r', 'a', 'n', 'g', 'e', 'm', 'e', 'l', 'o', 'n'] ret2 = re.findall("\w", "apple,banana,orange,melon")#['a', 'p', 'p', 'l', 'e', 'b', 'a', 'n', 'a', 'n', 'a', 'o', 'r', 'a', 'n', 'g', 'e', 'm', 'e', 'l', 'o', 'n'] ret2 = re.findall("\w+", "apple,banana,orange,melon")#['apple', 'banana', 'orange', 'melon'] ret2 = re.findall("\w+?", "apple,banana,orange,melon") # 取消贪婪匹配#['a', 'p', 'p', 'l', 'e', 'b', 'a', 'n', 'a', 'n', 'a', 'o', 'r', 'a', 'n', 'g', 'e', 'm', 'e', 'l', 'o', 'n'] ret2 = re.findall("\w*", "apple,banana,orange,melon")#['apple', '', 'banana', '', 'orange', '', 'melon', ''] ret2 = re.findall("\w{6}", "apple,banana,orange,melon")#['banana', 'orange'] #{}里面的6代表的是单词是六位的输出

【5】位置元字符**. ^ $**

正则 待匹配字符 匹配 结果 说明
海. 海燕海娇海东 海燕海娇海东 匹配所有"海."的字符
^海. 海燕海娇海东 海燕 只从开头匹配"海."
海.$ 海燕海娇海东 海东 只匹配结尾的"海.$"
```python import re # . 代表任意字符 pattern = "海."#检测海这个字 后面只能跟一个字符 letter = '''海燕海娇海东海冬梅''' res = re.findall(pattern, letter) print(res)# ['海燕', '海娇', '海东', '海冬'] # . 代表任意字符 # ^ 以 ... 开头 pattern = "^海."#以海开头，并且后缀一个字符 letter = '''海燕海娇海东海冬梅''' res = re.findall(pattern, letter) print(res)# # ['海燕'] # # . 代表任意字符 # # $ 以 ... 结尾 pattern = "海.$"#最后一个字节只能是海后面跟一个字节否则返回[] letter = '''海燕海娇海东海冬''' res = re.findall(pattern, letter) print(res)# ['海冬']

【7】字符集**［］［^］**

正则 待匹配字符 匹配 结果 说明
李[杰莲英二棍子]* 李杰和李莲英和李二棍子 李杰 李莲英 李二棍子 表示匹配"李"字后面[杰莲英二棍子]的字符任意次
李[^和]* 李杰和李莲英和李二棍子 李杰 李莲英 李二棍子 表示匹配一个不是"和"的字符任意次
[\d] 456bdha3 4 5 6 3 表示匹配任意一个数字，匹配到4个结果
[\d]+ 456bdha3 456 3 表示匹配任意个数字，匹配到2个结果
```python # # . 代表任意字符 # # [] 字符组中的任意一个 pattern = "李[杰莲英二棍子]"#李后面跟字符组中的任意一个 letter = '李杰李莲英和李二棍子' res = re.findall(pattern, letter) print(res)# ['李杰', '李莲', '李二'] # . 代表任意字符 # [] 字符组中的任意一个 # ^和 除了和 # * 任意 pattern = "李[^和]*"#[^和]：意思是不要"和"这个字 letter = '李杰李莲英和李二棍子' res = re.findall(pattern, letter) print(res)# ['李杰李莲英', '李二棍子']

【9】贪婪匹配

  • 贪婪匹配：在满足匹配时，匹配尽可能长的字符串，默认情况下，采用贪婪匹配
  • 非贪婪匹配：在满足匹配时，匹配尽可能短的字符串
  • 几个常用的非贪婪匹配，Pattern
```python import re # . 代表任意字符 pattern = "李.*?" #*? 重复任意次，但尽可能少重复 #['李', '李', '李'] pattern = "李.+?" #+? 重复1次或更多次，但尽可能少重复 #['李杰', '李莲', '李二'] pattern = "李.??" #?? 重复0次或1次，但尽可能少重复 #['李', '李', '李'] pattern = "李.{2,3}?" #{n,m}? 重复n到m次，但尽可能少重复 #['李杰李', '李二棍'] letter = '李杰李莲英和李二棍子' res = re.findall(pattern, letter) print(res)

(2)search（查找结果）

  • 只匹配到符合条件的一个字符
```python # 函数会在字符串内查找模式匹配,只到找到第一个匹配然后返回一个包含匹配信息的对象,该对象可以 # 通过调用group()方法得到匹配的字符串,如果字符串没有匹配，则返回None。 name_str = "Dream yuan Justin" pattern = 'a' result = re.search(pattern, name_str) print(result) # #span=(3, 4)意思是a的位置在3，4的位置 result = re.search(pattern, name_str).group() print(result) # 结果 : 'a'

(4)split（切割）

  • 先按’a’分割得到’‘和’bcd’,在对’'和’bcd’分别按’b’分割
```python name_str = 'abcd' # 先按'a'分割得到''和'bcd',在对''和'bcd'分别按'b'分割 pattern = '[ab]' result = re.split(pattern, name_str) print(result) # ['', '', 'cd']

(6)subn(全部替换)

  • 替换匹配到的所有字符并返回 替换成功的字符和替换次数
```python import re name_str = 'eva3egon4yuan4' # 将数字替换成'H'，返回元组(替换的结果,替换了多少次) result = re.subn('\d', 'H', 'eva3egon4yuan4') print(result) # ('evaHegonHyuanH', 3)
