requests模块

【一】Requests模块初始

【1】介绍

  • Requests 是⽤Python语⾔编写，基于 urllib ，采⽤Apache2 Licensed开源协议的 HTTP 库

  • 它⽐ urllib 更加⽅便，可以节约我们⼤量的⼯作，完全满⾜HTTP测试需求

  • 它用于发送HTTP请求

【2】来源

  • requests模块最初由Kenneth Reitz于2010年创建并开源，旨在提供一种更人性化的方式来发送HTTP请求

  • 可以模拟发送HTTP请求

【3】浏览器开发者模式：介绍

  • 元素 就是渲染在当前页面上的所有源代码

  • 控制台 可以控制当前页面文档对象和浏览器对象

  • 源代码 你自己的项目源码结构

  • 网络 捕获到所有发送和接收到的数据包 从网络入手

  • 应用 存储当前页面中的部分数据 Cookie / session

（2）方式二

  • 主动编码
  • 在浏览器中会默认将所有些参数进行URL编码—>urlencode
```python # 编码练习 from urllib.parse import urlencode keyword = { "wd": "周星驰" } url_encode = urlencode(keyword, encoding="utf8") print(url_encode) # 得到的是浏览器后面跟着的后面的乱码信息 # wd=%E5%91%A8%E6%98%9F%E9%A9%B0

（3）requests自动编码

  • requests模块自带一个帮助我们转码的参数:params
```python import requests from fake_useragent import UserAgent keyword = { "wd": "周星驰" } target_url = "https://www.baidu.com/" headers = { "User-Agent": UserAgent().random, } response = requests.get(url=target_url, headers=headers, params=keyword) response.encoding = "utf8" page_text = response.text print(page_text)

【3】模拟请求

```python from fake_useragent import UserAgent import requests target_url = "http://www.aa7a.cn/user.php" headers = { "User-Agent": UserAgent().random } data = { "username": "2870005304@qq.com", "password": "123456", "captcha": "skcc", "remember": "1", "ref": "http://www.aa7a.cn", "act": "act_login", } # 携带请求体参数有两种方式 response = requests.post( target_url, headers=headers, # 方式一 # data=data, # 方式二 json=data ) print(response.text) ```

（1）json=data

  • 当使用requests.post方式时
  • 将data参数设置为一个字典，并同时将headers参数中的Content-Type设置为application/json，那么data字典将被自动序列化为JSON字符串，并作为请求的主体数据发送
  • 这样的请求方式常用于与服务器交互时，需要使用JSON格式进行数据传输的情况

（2）data=data

  • 默认情况下，requests.post方法将会将data参数以application/x-www-form-urlencoded格式进行编码

  • 这种编码方式将字典数据转换成键值对的形式，并使用&符号进行连接

  • 然后，将生成的字符串作为请求的主体数据发送到服务器，这种方式常用于处理表单提交的场景
