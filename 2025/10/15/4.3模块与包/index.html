模块与包

【一】模块

【1】什么是模块

  • 在python中，一个py文件就是一个模块，文件名为xxx.py模块名则是xxx，导入模块可以引用模块中已经写好的功能。
  • 将程序模块化会使得程序的组织结构清晰，维护起来更加方便。
  • 使用模块既保证了代码的重要性，有增加了程序的结构性和可维护性。
  • 除了自定义模块外，我们还可以导入使用内置或第三方模块提供的现成功能，这种“拿来主义”极大地提高了程序员的开发效率。

【2】模块的来源

  • 内置的：python自带的，直接拿来用
  • 第三方的：别人写好的，直接拿来用，但是要先下载
  • 自定义的：就是自己写的

【3】模块的存在形式

  • 我们自己写的py文件（一个py文件就是一个模块）
  • 包：一系列py文件的集合。包也是一个文件夹
  • 一个包里面有一个.init.py文件

【4】模块的优点

  • 增加程序的健壮性和可读性
  • 大大增加了开发效率

【四】模块加载和搜索顺序

【1】模块的分类

  • 纯python编写的自己定义的模块
  • 内置的一系列模块 (python解释器自带的和你安装 pip install )
  • 使用c语言编写的底层代码
  • c 和 c++ 扩展出去的模块

【2】加载顺序

  • 先c 和 c++ 扩展出去的模块
  • 使用c语言编写的底层代码

  • 纯python编写的自己定义的模块

【五】绝对路径和相对路径

【1】相对路径

  • 你所在的文件夹就是相对路径

  • ./表示当前同目录下下载

  • …/表示当前目录的上一级目录下载

  • 默认的话就是在当前目录下

【2】绝对路径

  • 在python内部查找的顺序就是绝对路径
  • D:\Python\PythonProjects\PythonProjects29\day17
    • 就是带盘符的就是绝对路径

【六】包

【1】什么是包

  • 包是一个模块的集合，它可以将多个模块的功能组合在一起。
  • 包可以按照其功能和作用进行分类，方便用户查找和使用。
  • 包是在Python标准库中定义的一种特殊类型的模块，可以通过import语句引用和使用。
  • Python的包分为标准库包和第三方库包。
    • 标准库包是Python内置的包，包含了一些基本的模块和函数，如os、sys、random等；
    • 第三方库包是第三方开发的包，通常提供了更加丰富和高级的功能。

【2】如何创建包

（1）方法一

  • 创建一个文件夹

  • 在文件夹内部创建一个__init__.py文件

  • 这个文件夹就叫包

（2）方法二

  • 在pycharm右键文件新建package
  • 内部会自动生成__init__.py文件

【3】使用包

（1）方法一

#它们都是在一个文件夹内的

  • __init__.py#这个文本可以为空，但是必须要有
  • multy.py
  • main.py

(2)方法二

  • __init__.py#直接全部在本 文件文本中执行了
  • main.py

【七】导入语法总结

  • import 模块名 --》语法，只会检索当前模块内的所有变量名
  • import 包名–》语法，会先自动检索 init.py 文件内的所有变量名 然后再去检索 每个模块里面的变量名
    • 加载的时候多加载了一层 init.py
    • 查找的时候也多查找了一层 init.py
  • 不管是在模块还是包还是自己的文件中
  • from 模块名 import 属性名
  • from 包名 import 模块名
  • 只要检索到指定名字就会停止检索
