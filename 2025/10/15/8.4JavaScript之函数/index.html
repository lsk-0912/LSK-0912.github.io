【八】函数的数据类型

  • 函数是一种数据类型
  • 可以把函数作为另一个函数的参数，也可以把函数作为返回值
  • 赋值和传递
```javascript function user(name){ console.log(`My name is ${name}`) } var name=user; // name 变量现在引用了 user 函数 name("tom") // My name is tom // 将函数作为参数传递 function username(name,arg){ name(arg) } username(user,"jack"); // My name is jack

【九】arguments对象

【1】概括

  • js中arguments对象是一个特殊的对象，实际上是当前函数的一个内置属性
  • arguments对象：会接收所有的实参
  • length属性：可以获取参数的个数

【2】案例

```javascript function func(x,y,z){ console.log(arguments) console.log(x,y,z) } func(1,2,3,4,5,6,7,8,9) // 输出 // Arguments(9) [1, 2, 3, 4, 5, 6, 7, 8, 9, callee: ƒ, Symbol(Symbol.iterator): ƒ] // 1 2 3

【十】函数的递归

  • 函数的内部可以调用函数自己本身，这种做法称为递归
  • 函数递归需要考虑好结束递归的条件，不然会出现死递归导致程序崩溃，内存溢出
  • 递归一般是解决数学中的问题

【十一】参数和函数的作用域

  • 函数的参数是局部作用域，只能在函数内部使用
  • 函数本身就是一个独立的作用域

【十二】匿名函数

【1】介绍

  • 匿名函数又叫无名函数或lambda表达式，是没有名字的js函数定义形式

【2】语法

```javascript function(参数){代码体}

【4】将无名函数赋值给变量

```javascript var a = function() { console.log("Variable 'a' holds an anonymous function"); }; console.log(a()); Variable 'a' holds an anonymous function

【1】什么是箭头函数

  • 箭头函数（Arrow Function）是JavaScript ES6引入的一种新的函数定义语法，相比于传统函数表达式，它具有更简洁和易读的特点。
