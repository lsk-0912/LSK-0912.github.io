【7】axios的拦截器

  • 前言
````python # Axios 的拦截器是一种机制，允许你在请求发出之前或响应返回之后进行自定义处理。它分为请求拦截器和响应拦截器： # 请求拦截器：在请求被发送到服务器之前，你可以通过请求拦截器修改请求配置，如添加认证信息、修改请求头、处理请求数据等。 # 响应拦截器：在接收到服务器响应但在处理响应数据之前，你可以通过响应拦截器处理响应数据，如格式化响应数据、处理错误状态码、记录日志等。 # 使用拦截器可以帮助你统一处理请求和响应，提高代码的可维护性和一致性。

（2）请求拦截器

```js // 请求拦截器 // config 是一个对象，表示 HTTP 请求的配置信息 request.interceptors.request.use((config) => { // 从localStorage中取出token，如果有的话就携带 let token = localStorage.getItem('token'); // token是公共的参数，用户登录的需要携带参数 if (token) { // 因为我们用的是i第三方的token认证，我们也可以选择在配置文件中配置，不加Bearer config.headers.token = 'Bearer ' + token; } return config; })

（3）请响应拦截器

  • 简约的
````js // 响应拦截器 instance.interceptors.response.use( (response) => { // 对响应数据做些什么 return response; }, (error) => { // 对响应错误做些什么 return Promise.reject(error); } ); export default request

（4）总和

````js import axios from "axios"; import router from "../router"; import Element from "element-ui" // axios.defaults.baseURL = "http://localhost:8081" axios.defaults.baseURL = "http://127.0.0.1:8001/api/v1" //创建请求对象 //指定超时时间为5000毫秒 //指定请求头中编码格式为application/json const request = axios.create({ timeout: 5000, headers: { 'Content-Type': "application/json; charset=utf-8" } }) // 请求拦截器 // config 是一个对象，表示 HTTP 请求的配置信息 request.interceptors.request.use((config) => { // 从localStorage中取出token，如果有的话就携带 let token = localStorage.getItem('token'); // token是公共的参数，用户登录的需要携带参数 if (token) { // 因为我们用的是i第三方的token认证，我们也可以选择在配置文件中配置，不加Bearer config.headers.token = 'Bearer ' + token; } return config; }) //在请求响应后进行拦截， // 如果响应数据中的status为100，表示成功，不做处理，如果不是100，使用elementui显示错误信息 // 如果响应对象的状态码为401，重定向到登陆页面，只要有错误，就打印错误信息 request.interceptors.response.use( response => { console.log("response ->" + response) let res = response.data if (res.code === 100) { return response.data } else { Element.Message.error(!res.msg ? '请求服务器异常,请联系管理员' : res.msg) return Promise.reject(response.data.msg) } }, error => { console.log(error) if (error.response.data) { error.massage = error.response.data.msg } if (error.response.status === 401) { router.push("/login") } Element.Message.error(error.massage, {duration: 3000}) return Promise.reject(error) } ) export default request
