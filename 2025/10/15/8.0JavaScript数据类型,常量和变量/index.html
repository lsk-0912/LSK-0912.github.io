JavaScript数据类型,常量和变量

【一】调试语法

【1】alert：警告–>弹框

  • 在浏览器中弹出一个警告框，内容为alert里面的内容
```javascript alert(内容); alert("Surprise");

【3】console：控制台–>print()

  • 在控制台内输出一些内容
```javascript closole.log(内容); console.log("Surprise");
  • null的数据类型是object
```javascript typeof(null); 'object'
  • var：在循环内定义也会影响全局
  • let：在局部内定义只会影响局部
    • 编译器版本是 5.1 则无法使用let

（2）命名规范

  • 数字/字母/下划线/$
  • 变量名命名规范(js中推荐驼峰式命名法、少有成员以_开头)
  • 变量名不能用关键字来声明
  • 变量名不能用数字开头

（3）变量赋值

  • 定义变量后没有赋值，内部会有一个默认的存储值叫做undefined
```javascript var id; id; undefined

【2】常量

  • js也是一门面向对象的编程语言，一切皆对象

  • const：定义常量

```javascript const pi=x;

【四】数值类型–>Number

【1】定义数值，查看类型

```javascript var age=18; typeof age; 'number' var money=1200.31; typeof money; 'number'
  • 浮点数转浮点数用parseFloat方法
```javascript var money="1200.31"; typeof money; 'string' var money_one=parseFloat(money); typeof money_one; 'number'

【2】书写多行字符串

```javascript var s3 = ` as df ce gh ` typeof s3; 'string

【4】拼接字符

  • 用加号拼接
```javascript var name="jack"; var age=18; name+age; 'jack18'
  • 移除空白：.trim()
```javascript var name=" tom "; name.trim(); 'tom' name ' tom '
  • 移除右边的空白：.trimRight()
```javascript name.trimRight(); ' tom'
  • 拼接：.concat(value,…)
```javascript name.concat(age); ' tom 18'
  • 根据索引获取子序列：.substring(from,to)
```javascript var name="nametomjack"; name.substring(-1,4); 'name'
  • 小写：.toLowerCase()
```javascript name.toLocaleLowerCase(); 'nametomjack'
  • 分割：.split(delimiter,limit)
```javascript var name="nametomjack"; name.split("t"); ['name', 'omjack']
  • 方法.toString()
```javascript let num=123; let num_one=num.toString(); typeof num_one; 'string'

【2】Boolean是false的如下

  • 空字符串

  • 0

  • null

  • undefined

  • NaN

```javascript Boolean(NaN); false Boolean(null); false Boolean(0); false Boolean(undefined); false Boolean(); false

【七】null和undefined的区别

【1】null

  • 表示值为空
  • 指定或清空变量的时候会用

【2】undefined

  • 表示只是声明了变量并没有赋值
