JavaScript数据类型,常量和变量

【一】调试语法

【1】alert：警告–>弹框

  • 在浏览器中弹出一个警告框，内容为alert里面的内容

【2】prompt：对话框–>input

  • 第一个参数是询问内容，第二个参数是提示摊入的实例值

【3】console：控制台–>print()

  • 在控制台内输出一些内容

【二】数据类型

【1】数据类型概览

  • 简单数据类型：Number，String，undefined，Boolean，null
  • 复杂数据类型：Object

【2】数据类型检测

  • 用typeof方法
  • null的数据类型是object

【3】字面量

  • 字面量是用于表达固定值的方法，又叫常量
  • 所见即所得
  • 比如：数字、字符串、布尔值等

【4】数字字面量

  • 这里指数学意义上的数字
  • 数字字面量区分：整数、浮点数、特殊值
  • 书写时直接写字面量，不需要加任何符号

【5】特殊值—>number类型

  • Infinity无穷：无穷大Infinity，无穷小-Infinity
  • NaN: 不是一个正常的数，是数字类型，可以通过isNaN来判断

【6】字符串

  • 组成：字母，汉字，特殊符号，空白等

  • 字符串的字面量写法：是用一对单引号或者一对双引号

  • 如果字符串中含有单引号那么字面量写法就必须用双引号，反之也一样

  • 转义符号：\n换行 ，\t缩进 ，’ 单引号 ，" 双引号，\反斜杠

【三】常量和变量

【1】变量

（1）声明变量

  • name可以指向任意类型
  • var：在循环内定义也会影响全局
  • let：在局部内定义只会影响局部
    • 编译器版本是 5.1 则无法使用let

（2）命名规范

  • 数字/字母/下划线/$
  • 变量名命名规范(js中推荐驼峰式命名法、少有成员以_开头)
  • 变量名不能用关键字来声明
  • 变量名不能用数字开头

（3）变量赋值

  • 定义变量后没有赋值，内部会有一个默认的存储值叫做undefined
  • 同时赋值多个变量

【2】常量

  • js也是一门面向对象的编程语言，一切皆对象

  • const：定义常量

  • 案例
    • 定义了常量改了就会报错

【四】数值类型–>Number

【1】定义数值，查看类型

【2】注意NaN

  • 特殊的NaN：数值类型
  • 表示的不是一个数字

【3】类型转换

  • 用parselnt方法
  • 浮点数转浮点数用parseFloat方法

【4】特殊值NaN

  • NaN是整数类型，但是他不是一个数字，不可以相互加减

  • 加减后的结果就是NaN

【五】字符串类型–>String

  • 尽量用""双引号

【1】定义字符串，查看类型

  • string可以先声明不赋值

【2】书写多行字符串

【3】格式化字符串

  • 书写${}会向前找之定的变量值
    • 找不到就会报错

注意：这里用的不是单引号而是数字1旁边的`符号

【4】拼接字符

  • 用加号拼接

【5】常用方法

  • 返回长度：.length
  • 移除空白：.trim()
  • 移除左边的空白：.trimLeft()
  • 移除右边的空白：.trimRight()
  • 返回第n个字符：.charAt(n)
    • 索引位置从0开始
  • 拼接：.concat(value,…)
  • 子序列位置：.indexOf(substring,start)
    • 用于返回指定值（子字符串）在字符串中首次出现的位置。如果未找到该值，则返回 -1。
  • 根据索引获取子序列：.substring(from,to)
  • 切片：.slice(start,end)
  • 小写：.toLowerCase()
  • 大写：.toUpperCase()
  • 分割：.split(delimiter,limit)

【6】转换类型

  • 方法.string()
  • 方法.toString()

【六】布尔类型–>Boolean

【1】语法演示

  • 在js中bool值是全小写的

【2】Boolean是false的如下

  • 空字符串

  • 0

  • null

  • undefined

  • NaN

除了上述为 false 的情况，其他情况均为 true

【七】null和undefined的区别

【1】null

  • 表示值为空
  • 指定或清空变量的时候会用

【2】undefined

  • 表示只是声明了变量并没有赋值
