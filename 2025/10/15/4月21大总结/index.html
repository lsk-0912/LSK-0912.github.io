4.21大总结

【一】函数

  • 什么是函数

    • 就是把一个复杂的需求拆分，然后做成单个功能的，在全局都能用（前提是在调用函数的下面使用）

  • 函数的标志-----》 def打头

  • 函数的调用方式

    • 直接掉调用
      • def name_age(name,age)#调用的时候别忘记给参数
    • 表达式调用
      • data = age(2,3) + 6#age()是函数
    • 函数作返回值
      • 函数的返回值作为参数，给其他参数使用
      • 函数的内存地址可以作为参数

  • 函数的形参

    • 就是在定义函数（）里面的参数名

  • 函数的实参

    • 就是函数被调用阶段进去的参数

  • 传参

    • 位置传参
    • 关键字传参
    • 默认值传参

  • 可变长位置擦参数

    • *ages

  • 可变长关键字参数

    • **kwargs

【二】名称空间

  • 内建名称空间：python自带的

  • 全局名称空间：自己定义的

  • 局部名称空间：定义在函数/类 里面的函数名…

  • 程序启动的加载顺序–

    • 先内-再全-最后局

  • 寻找变量名的加载顺序

    • 先局-在全-最后内

【三】作用域

  • 局部作用域（Local）

    • 函数亡，局部变量亡
    • 范围：整个函数体范围
    • 查看局部变量：locals

  • 嵌套作用域(Enclosed)

    • 范围：闭包函数

  • 全局作用域（Global）

    • 在模块文件最顶层生命的变量，具有全局性
    • 修改全局变量我们可以用globals

  • 内建作用域（Built）

    • 随着python启动自己加载的名称空间

  • 查找顺序legb

  • 全局变量和局部变量重名时，采用就近原则

【四】函数对象

  • python中一切皆对象
  • 函数可以直接被引用
    • 定义一个函数用一个新的变量来存，用新的变量名调用
  • 可以做为元组被储存
  • 函数可以作为一个参数传给另一个参数
  • 函数的返回值可以是一个参数传给另一个参数
  • 函数的返回值可以是一个函数
    • 直接将函数的内存地址返回

【五】装饰器

  • 在不修改被装饰对象源代码和调用方式的前提下为被装饰对象添加额外的功能。
  • 分类
    • 有参函数
    • 无参函数
  • 原理
    • 闭包函数和函数对象的组合使用
  • 无参装饰器
    • 没有参数的装饰器
  • 模板
```python # 无参装饰器的模板 def outer(func): ''' :param func: 每一次需要调用传递的函数 :return:返回值是内嵌函数 inner 的函数内存地址 ''' def inner(): # 可以加逻辑校验 # 当符合指定条件后，可以允许执行传递进来的函数 # 不符合条件的时候不执行函数 func() # 做逻辑校验，func 函数执行后的逻辑 return inner

  • 装饰器语法糖

  • 标志@

  • 无参装饰器语法糖

    • 无参数

  • 有参语法糖

    • 有参数
    • 但语法糖：就@
    • 多语法糖：多个@

【六】迭代器

  • 每次迭代得到的返回值结果可就是下一次迭代开始的初始值
  • 可迭代对象
    • 内置有.__iter__方法的对象都是可迭代对象
    • 不可迭代的：布尔，整数，浮点数，剩下都的可以

【七】迭代器对象

  • 内置有.__iter__和.__next__就是迭代器对象

  • .__next__挨个取值一次只能取一个，下次从你上一次的取值结果继续开始

  • 也可以直接写成next（），iter（）

  • 缺点：除非取值取到尽头，否则永远不知道终止索引在哪里

    • 只能取一次记一次位置，下一次的起始位置是上一次的终止位置，想回到开头，回不去

【八】生成器

  • 在python中生成器就是一种特殊的迭代器

  • 创建方式

    • 可以将列表推导式的方括号改为圆括号，即可创建一个生成器

    • yield关键字

      • 生成器函数中的yield语句会暂停函数执行并返回一个值，下一次调用该函数时会继续执行并返回下一个值
      • send（）传值

【九】模块与包

  • 模块

    • 一个py文件就是一个模块

  • from 模块名 import 属性名

  • import 模块名

  • 支持导入多个模块

  • 重命名模块—》import 模块名 as 新模块名

  • 模块分类

    • 纯python编写的自己定义的模块
    • 内置的一系列模块 (python解释器自带的和你安装 pip install )
    • 使用c语言编写的底层代码
    • c和c++扩展出来的模块

  • 加载顺序

    • 跟上面反过来

  • 查找顺序

    • 跟加载顺序相反

  • 包

    • 模块的聚集地
    • 里面有.__init__文件的就是包

  • from 模块名 import 属性名

  • from 包名 import 模块名

【十】绝对路径

  • 在python查找的顺序就是绝对路径
    • D:\Python\PythonProjects\PythonProjects29\day17

【十一】相对路径

  • 你所在的文件夹就是相对历经
  • ./当前同目录下载
  • .//当前目录上一级目录下载
  • 默认就是当前目录下载

【十二】json模块

  • json是 你存进去是什么类型读出来就是什么类型

  • 导入模块

    • import json模块

  • 保存为json文件

    • with open…
    • 写入用json.dump(存数据的变量名，句柄= 句柄)

  • 读出来转为python对象

    • with open…
    • 读出用json.load(句柄)

  • 将字典转换为字符串

    • json.dumps(变量名)

  • 字符串转换为字典

    • json.loads

【十二】time模块

  • 生成时间戳：time.time()

  • time.sleep():睡眠时间

  • 生成时间戳转为时间元组

    • 国际时间time.gmtime(time.time)
    • 本地时间time.localtime(time.time)

  • 转换为时间字符串time.strftime(‘%Y-%m-%d %H:%M:%S’)

  • 从时间元组中获取指定的元素

    • 年：变量名.tm_year
    • 月：变量名.tm_mon
    • …

  • time.strptime(“2023-11-24”, “%Y-%m-%d”)括号里可以改任意时间

    • 将时间字符串格式转换为时间元组
