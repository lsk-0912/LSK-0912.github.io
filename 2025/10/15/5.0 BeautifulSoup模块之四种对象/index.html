BeautifulSoup模块之四种对象

【一】前言

  • 在生成的soup中存在四种取值方式

    • BeautifulSoup

    • Tag

    • NavigableString

    • Comment

【二】介绍四种对象

【1】BeautifulSoup对象

  • 代表解析后的HTML文档，是最顶层的对象
  • 它包含了整个文档的全部内容，并提供了操作HTML文档的方法和属性
```python from bs4 import BeautifulSoup # 假设html_doc是一个包含HTML内容的字符串 html_doc = """ The Dormouse's story

The Dormouse's story

Once upon a time there were three little sisters; and their names were Elsie, Lacie and Tillie; and they lived at the bottom of a well.

...

""" # 使用html.parser解析器创建一个BeautifulSoup对象 soup = BeautifulSoup(html_doc, 'html.parser')

（1）查看tag对象

  • 通过soup.tag获取到指定标签对象
```python from bs4 import BeautifulSoup # 读取html文档 with open("练习.html", "r", encoding="utf") as fp: data = fp.read() # 生成soup对象 soup = BeautifulSoup(data, "lxml") # 指定标签名查找指定的标签对象 # 输出head标签及其属性 head = soup.head print(head, type(head)) # # 输出title标签及其类型 title = soup.title print(title, type(title)) # # 输出第一个a标签和类型 a = soup.a print(a, type(a)) # # 输出第一个p标签下的第一个a标签 b = soup.p.b print(b, type(b)) #

（3）返回多值的属性

  • 常见的多值的属性是 class (一个tag可以有多个CSS的class)
  • 在BeautifulSoup多值属性的返回类型都是list
```python from bs4 import BeautifulSoup # 读取html文档 with open("练习.html", "r", encoding="utf") as fp: data = fp.read() # 生成soup对象 soup = BeautifulSoup(data, "lxml") # 返回当前标签的class属性值 print(soup.a["class"], type(soup.a["class"])) # ['sister']

【3】NavigableString对象

  • 表示标签内的文本内容，或者是非标签字符串
  • 当tag只包含单一的字符串时，可以使用tag.string或tag.text来获取该字符串
```python from bs4 import BeautifulSoup # 生成soup对象 with open("练习.html", "r", encoding="utf8") as fp: data = fp.read() soup = BeautifulSoup(data, "lxml") # 或群单一标签的文本内容 text = soup.title.string print(text) # 获得p标签下所有的文本信息 p_text = soup.p.strings print(p_text) # The Dormouse's story # for p in p_text: print(p) # The Dormouse's story
